
Available SchemaCrawler catalog loaders:
 weakassociationsloader
                Loader for weak associations
 testloader     Loader for testing
 attributesloader
                Loader for catalog attributes, such as remarks or tags
 countsloader   Loader for table row counts
 schemacrawlerloader
                Loader for SchemaCrawler metadata catalog

Notes:
- For help on an individual catalog loader,
  run SchemaCrawler with options like: `-h loader:weakassociationsloader`
  or, from the SchemaCrawler interactive shell: `help loader:weakassociationsloader`
- Options for the catalog loaders should be provided with the `load` command
